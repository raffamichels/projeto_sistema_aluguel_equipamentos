# 01 Definição das Entidades Chave (Modelo RentEasy - 12 Tabelas)

## Escopo do Projeto: Aluguel de Equipamentos (RentEasy)

O modelo físico é projetado com 12 tabelas para garantir a robustez e a complexidade necessária para atender a todos os requisitos do trabalho (Normalização, 4 Constraints, Objetos e 6 Perguntas de Negócio).

---

## I. Resumo das 12 Entidades (Modelo Físico)

| Tabela | Propósito e Ligação Principal |
| :--- | :--- |
| **1. Clientes** | Dados cadastrais dos clientes que alugam. |
| **2. Cargos** | Tabela de apoio para definir os cargos dos Funcionários. |
| **3. Funcionários** | Dados da equipe interna que processa os aluguéis. |
| **4. Enderecos** | Endereços completos, isolados para uso por Clientes ou Logística. |
| **5. Categorias** | Classificação dos Equipamentos. |
| **6. Equipamentos** | Dados mestres do produto (Modelo, Fabricante). |
| **7. Estoque** | **Unidades Físicas** do equipamento (N° de Série, Status). |
| **8. Alugueis** | Transação principal (Pedido, Datas de Início/Fim, Valor Total). |
| **9. Itens\_Aluguel** | **Relação N:N**. Liga a transação de Aluguel ao item específico do Estoque. |
| **10. Pagamentos** | Registro de todos os pagamentos e métodos de pagamento. |
| **11. Manutencoes** | Registra o histórico e custos de manutenção dos itens do Estoque. |
| **12. Multas** | Registra multas aplicadas por atraso ou danos ao equipamento alugado. |

---

## II. Detalhamento do Modelo Físico (Entidade e Colunas)

### 1. Clientes

| Coluna | Chave | Tipo Sugerido | Descrição |
| :--- | :--- | :--- | :--- |
| **ID\_Cliente** | **PK** | INT | Chave Primária |
| **Nome** | | VARCHAR(100) | |
| **Tipo\_Pessoa** | **CHECK** | CHAR(1) | 'F' (Física) ou 'J' (Jurídica) |
| **CPF\_CNPJ** | **UNIQUE** | VARCHAR(18) | |
| **Email** | | VARCHAR(100) | Login no Front-end (RentEasy) |
| **Telefone** | | VARCHAR(15) | |
| **ID\_Endereco** | **FK** | INT | Endereço principal do cliente |
| **Data\_Cadastro** | **DEFAULT** | DATE | Data de entrada no sistema |

### 2. Cargos

| Coluna | Chave | Tipo Sugerido | Descrição |
| :--- | :--- | :--- | :--- |
